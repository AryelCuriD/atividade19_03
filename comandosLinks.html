Comandos e Links


Comandos principais(Minecraft: Java Edition):

Comandos Descrição
/advancement Concede, remove ou verifica os progressos de um jogador.
/attribute Consulta, adiciona, remove ou define um atributo de uma entidade.
/ban Adiciona um jogador para a lista preta.
/ban-ip Adiciona um endereço IP para a lista preta.
/banlist Exibe a lista preta.
/bossbar Cria e modifica barras de chefe
/clear Exclui os itens do inventário do jogador
/clone Copia blocos de um lugar a outro
/damage Aplica dano às entidades especificadas
/data Recebe, funde, modifica e remove blocos-entidade, entidades e dados NBT de armazenamento de comandos.
/datapack Controla pacotes de dados carregados.
/debug Inicia ou para uma seção de depuração.
/defaultgamemode Define o modo de jogo padrão.
/op /deop Dá ou retira as permissões de operador de um jogador.
/difficulty Define o nível de dificuldade.
/effect Adiciona ou remove efeitos.
/enchant Adiciona um encantamento a um item selecionado pelo jogador.
/execute Executa outro comando.
/experience Outro nome de /xp. Adiciona ou remove a experiência de um jogador.
/fill Preenche uma região com um certo bloco.
/fillbiome Preenche uma região com um certo bioma.
/forceload Força os chunks a serem constantementes atualizados ou não.
/function Carrega uma função.
/gamemode Define o modo de jogo de um jogador.
/gamerule Define ou verifica um valor de regra de jogo.
/give Concede um item a um jogador.
/help Provê ajuda para comandos
/item Manipula itens em inventários.
/jfr Inicia ou para um perfil JFR.
/kick Expulsa um jogador de um servidor.
/kill Mata entidades (jogadores, criaturas, itens, etc.)
/list Lista jogadores no servidor.
/locate Localiza a estrutura, bioma ou ponto de interesse mais próximo.
/loot Solta itens de um espaço de inventário no chão.
/me Exibe uma mensagem sobre o remetente.
/msg Um alias de /tell e /w. Exibe uma mensagem privada para outros jogadores.
/pardon Remove nomes da lista preta.
/pardon-ip Remove endereços IP da lista preta.
/particle Cria partículas.
/perf Grava informações e métricas sobre o jogo por 10 segundos.
/place Usado para colocar um recurso configurado, quebra-cabeça, predefinição ou estrutura em um local dado.
/playsound Toca um som.
/publish Abre um mundo de Um Jogador para a rede local.
/random Extrai um valor aleatório ou controla a sequencia de números aleatórios.
/recipe Concede ou remove receitas de jogador.
/reload Recarrega tabelas de saque, progressos e funções no disco.
/return Controla sequencia de execução dentro de funções e muda seus valores devolvidos.
/ride Usado para fazer entidades montarem em outras entidades, parar as entidades de montarem, fazer montarias ejetarem seus montadores, ou invocar montarias ou montadores.
/rotate Muda a rotação de uma entidade.
/save-all Salva o servidor ao disco.
/save-off Ativa salvas automáticas do servidor
/say Exibe uma mensagem para vários jogadores.
/schedule Atrasa a execução de uma função.
/scoreboard Gerencia objetivos de placares e jogadores.
/seed Exibe a semente do mundo.
/setblock Muda um bloco para outro bloco.
/setidletimeout Define o tempo que um jogador precise ficar ocioso para ser expulso.
/setworldspawn Define a geração do mundo.
/spawnpoint Define o ponto de geração de um jogador.
/spectate Faz com que um jogador especte uma entidade.
/spreadplayers Teleporta entidades para locais aleatórios.
/stop Fecha um servidor.
/stopsound Para um som.
/summon Invoca um entidade.
/tag Controla as marcações de entidade
/team Controla times.
/teammsg Um alias de /tm. Especifica a mensagem para enviar ao time
/teleport Um alias de /tp. Teleporta entidades.
/tell Um alias de /msg e /w. Exibe uma mensagem privada a outros jogadores.
/tellraw Exibe uma mensagem JSON para jogadores.
/test Gerencia e executa GameTests.
/tick Controla a taxa de ticks em um mundo.
/time Muda ou verifica o tempo de jogo de um mundo.
/title Gerencia telas-título
/tm Um alias de /teammsg. Especifica a mensagem para enviar ao time.
/tp Um alias de /teleport. Teleporta entidades.
/transfer Inicia a transferência de uma jogador para outro servidor.
/trigger Define um gatilho a ser ativado
/w Um alias de /tell and /msg. Exibe uma mensagem privada a outros jogadores.
/weather Define o clima.
/whitelist Gerencia a lista branca de um servidor.
/worldborder Gerencia a barreira do mundo
/xp Um alias de /experience. Adiciona ou remove a experiência de um jogador.
